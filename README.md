# Каталог продукции для компании

## Дипломный проект
- Студент: Александров Егор Александрович
- Группа: 15.ИСиП.22.О-ЗФ.С.2.ХК
- Тема диплома: Разработка каталога продукции для компании

## О проекте
Веб-приложение каталога продукции с корзиной, регистрацией пользователей, оформлением заказов и административной панелью для управления каталогом и остатками товара.

## Технологии
- Frontend: HTML5, CSS3, JavaScript (Vanilla JS)
- Backend: PHP 8
- База данных: SQLite
- Веб-сервер: Apache2

## Сервер и размещение
- Для дипломной работы арендован виртуальный сервер на FirstVDS.
- Все работы выполнялись напрямую на сервере через Visual Studio Code.
- Публичный адрес сайта: [https://aleksandrov-egor.ru](https://aleksandrov-egor.ru)

## Как проверить работу сайта
1. Откройте сайт по ссылке: [https://aleksandrov-egor.ru](https://aleksandrov-egor.ru)
2. Перейдите в `Кабинет` и зарегистрируйте нового пользователя:
- Укажите имя, телефон и пароль.
- После регистрации выполнится автоматический вход.
3. Перейдите в `Каталог`, выберите товар и количество, добавьте в корзину.
4. Оформите заказ в `Корзине`.
5. Проверьте, что остатки товара уменьшаются после оформления заказа.
6. Для проверки админ-функций войдите под тестовым админ-аккаунтом и откройте `admin.php`:
- Измените поле `В наличии` у товара.
- Нажмите `Сохранить изменения`.
- Убедитесь в каталоге, что значение обновилось.

## Локальный запуск (для проверки)
1. Установите PHP 8+.
2. Из корня проекта запустите:

```bash
php -S localhost:8000
```

3. Откройте: `http://localhost:8000`
